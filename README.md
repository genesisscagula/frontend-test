# Frontend Test Implementation

This project is a frontend implementation based on the provided design requirements.  
The goal of this implementation is to recreate the UI while maintaining a clean and organized code structure using plain HTML, CSS, and JavaScript.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

project-folder
│
├── docs
│   ├── index.html
│   ├── css
│   │   └── style.css
│   ├── js
│   │   └── script.js
│   └── images
│
└── README.md

The `docs` folder contains the frontend files used to render the UI.

## Features

- Responsive layout
- Clean and readable code structure
- Vanilla JavaScript for basic interactivity
- Organized CSS for maintainability

## Setup Instructions

1. Clone the repository

git clone https://github.com/genesisscagula/frontend-test.git

2. Open the project folder

3. Open `docs/index.html` in your browser
